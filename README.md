# Prashant Yadav Portfolio (Next.js)

This project converts the provided resume into:
- A portfolio website
- Project-focused blog posts
- A dedicated ANPR case study page

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Routes

- `/` Portfolio homepage
- `/blog` Blog index
- `/blog/[slug]` Individual blog post pages
- `/case-studies/anpr-system` ANPR case study
